# Модификаторы доступа
- `public`:
  - Для класса: доступен из любого места;
  - Для члена класса: доступен из любого места.
- `protected`:
  - Для члена класса: доступен в самом классе и его наследниках.
- `private`:
  - Для класса: доступен только для вложенных классов (класс внутри класса).
  - Для члена класса: доступен только внутри класса.
