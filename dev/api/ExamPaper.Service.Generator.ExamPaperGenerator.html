Настройки генерации, определяющие количество билетов и вопросов в каждом билете.

Returns

IEnumerable<IExamPaper>

Коллекция сгенерированных экзаменационных билетов.

Remarks

Вопросы для каждого билета выбираются случайным образом без повторений в пределах одного билета. Один и тот же вопрос может встречаться в разных билетах.

Exceptions

ArgumentNullException

Выбрасывается, если availableQuestions или settings равны null.

InvalidOperationException

Выбрасывается, если количество доступных вопросов меньше, чем QuestionsPerTicketCount.
